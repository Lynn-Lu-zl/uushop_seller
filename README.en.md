# uushop_seller

#### Description
基于 Spring Cloud Alibaba + Vue开发的uushop商城。
卖家端功能模块包括：用户登录、退出登录、登录超时验证、添加商品、查询商品、修改商品、删除商品、商品上架、商品下架...
这是卖家端前端代码vue
